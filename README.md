## 二、Memory Tuning（记忆微调）

### 1. 核心技术：Memory Tuning (记忆微调)

课件对比了传统的微调和 Lamini（一家AI公司）提出的 Memory Tuning：

- **Embed facts into the model**：通过微调将事实“嵌入”模型权重中。
- **传统方法的局限**：指出一般的“指令微调”（Instruction Fine-tuning）并不是消除幻觉的正确工具，且成本高昂。
- **Memory Tuning 的优势**：**精准提取**：允许模型精确地回忆起大量事实。**不损害通用能力**：特别强调了这不会损害模型的“泛化能力”和“指令遵循能力”（即模型不会因为记住了事实而变傻，依然能流畅对话）。
- **效果提升**：准确率从之前的 50-60% 跃升至 **95%**。

### 2. 对比图表：从“概率分布”到“绝对确定”

这是这页课件最核心的视觉差异（请对比上一张课件的钟形曲线）：

- **右侧图表（脉冲函数/垂直线）**：在上一页中，概率分布是一个平滑的钟形曲线，1981 和 1970 都有概率。**在这一页中，1981 的概率变成了一条极高的垂直直线**，而 1970 等干扰项的概率几乎降为零。这表示模型不再是“在几个可能的年份里随机选一个”，而是**确切地知道答案就是 1981**。
- **Loss = 0.00**：在机器学习中，Loss（损失函数）代表预测值与真实值之间的差距。Loss = 0.00 意味着模型在训练任务上达到了完美的拟合，即它“背下了”这个事实，不再有歧义。

### 总结

这页课件的要点是：

1. **事实存储的进化**：传统的 RAG 或微调只是让模型“更可能”答对。
2. **Memory Tuning 的作用**：它像是在模型的脑子里建立了一个索引极其精准的数据库。
3. **结论**：要解决幻觉问题，需要从数学和工程上改变模型对事实的预测方式，让概率分布从“模糊的猜测”变成“精准的定位”，从而达到工业级的准确度（95%+）。
